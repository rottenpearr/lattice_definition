# Информация по проекту

## Историческая справка

Хронологическое развитие проекта:
1. База данных;
2. Математическое совпадение типов кристаллов;
3. Поиск микрокристаллов в макрокристаллах;
4. Колебания/вакансии кристаллов;
5. Механический аспект распознавания + автоматизация;
6. Машинное обучение;
7. Сопоставление с альтернативными способами/патентование.

Распределение задач по темам дипломов:
- Артюшин: 3 + 5 + 6 + что-то еще (работа с патентом)
- Маркова: 1 + 7 + поисковик
- Черняков: 2 + 4 + своя нейронка + визуализация 3D

Темы курсовых работ по дисциплине "Базы данных":
1. Маркова: `РАЗРАБОТКА СУБД ДЛЯ КЛАССИФИКАЦИИ И РАСПОЗНАВАНИЯ ТИПОВ КРИСТАЛЛИЧЕСКИХ РЕШЕТОК`
2. Черняков: `ОРГАНИЗАЦИЯ ИМПОРТА ДАННЫХ ИЗ CIF-ФАЙЛОВ В СУБД «КРИСТАЛЛИЧЕСКИЕ РЕШЕТКИ»`

Темы дипломных работ (первичные названия):
1. Артюшин: `РАЗВИТИЕ СИСТЕМЫ РАСПОЗНАВАНИЯ ТИПОВ КРИСТАЛЛИЧЕСКИХ РЕШЕТОК С ПРИМЕНЕНИЕМ МЕТОДОВ РАДИАЛЬНОГО РАСПРЕДЕЛЕНИЯ БЛИЖАЙШИХ СОСЕДЕЙ И МАШИННОГО ОБУЧЕНИЯ`
2. Маркова: `ВНЕДРЕНИЕ МЕХАНИЗМОВ РАСПОЗНАВАНИЯ ТИПОВ КРИСТАЛЛИЧЕСКИХ РЕШЕТОК ДЛЯ АВТОМАТИЗАЦИИ ОПРЕДЕЛЕНИЯ МЕТОДОВ МОЛЕКУЛЯРНО-ДИНАМИЧЕСКИХ ИССЛЕДОВАНИЙ`
3. Черняков: `РАЗВИТИЕ СИСТЕМЫ ГЛОБАЛЬНОГО РАСПОЗНАВАНИЯ ТИПОВ КРИСТАЛЛОВ С ПРИМЕНЕНИЕМ МЕТОДОВ AI/ML ДЛЯ ЧИСЛЕННОГО МАТЕРИАЛОВЕДЕНИЯ`
