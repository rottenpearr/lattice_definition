UO₂ — диоксид урана, химическое соединение урана(IV), обладающее кубической структурой типа флюорита (CaF₂). Твёрдое вещество чёрного цвета с высокой температурой плавления. Основное топливо для ядерных реакторов.
Применение
Ядерное топливо в реакторах на тепловых нейтронах, производство MOX-топлива, научные исследования.
